Class: Enumeration

Enumeration(IRI, graph)

new Enumeration(IRI, graph)

An Enumeration represents a schema:Enumeration, which is also a sub-type of an rdfs:Class. It is identified by its IRI
Parameters:
Name Type Description
IRI string The compacted IRI of this Enumeration, e.g. "schema:DayOfWeek"
graph object The underlying data graph to enable the methods of this Enumeration
